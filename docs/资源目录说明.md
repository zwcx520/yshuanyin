# 官网资源目录说明

## images

存放网站图片资源。

```text
images/
├── logo/      公司 Logo、品牌标识
├── banners/   首页和内页大图
├── artists/   艺人照片
├── albums/    专辑封面
├── events/    演出活动海报和现场图
└── news/      新闻封面图
```

当前页面已经使用：

1. `images/banners/`
2. `images/artists/`
3. `images/albums/`
4. `images/news/`

## audio

存放音乐试听文件，建议使用 `mp3` 格式。

## video

存放视频素材，建议使用 `mp4` 格式，并控制文件大小。

## vendor

存放本地第三方组件库，包括 Bootstrap、Bootstrap Icons、Swiper 和 AOS。

## 替换素材注意事项

1. 替换图片时尽量保持文件名不变，页面无需修改。
2. 艺人照片建议使用竖版图。
3. 专辑封面建议使用正方形图。
4. 新闻封面和 Banner 建议使用横版图。
5. 音频和视频上线前需要确认版权和播放授权。
